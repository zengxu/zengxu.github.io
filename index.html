Input mathematical formula in hexo

Posted on 2016-01-23   |  

Install Hexo-math plugins as following:

$ npm install hexo-math --save

then, execute the following scripts in the hexo folder:

$ hexo math install

Finally, in the _config.yml file of the system, add:

plugins:
 - hexo-math

Instructions for use:

Insert the formula in a line: begin with “\$” and end with “\$”
Insert the formula independently: begin with “\$\$” and end with “\$\$”

Examples:

\[J\alpha(x) = \sum{m=0}^\infty \frac{(-1)^m}{m! \Gamma (m + \alpha + 1)} {\left({ \frac{x}{2} }\right)}^{2m + \alpha}\]

$$[J\alpha(x) = \sum{m=0}^\infty \frac{(-1)^m}{m! \Gamma (m + \alpha + 1)} {\left({ \frac{x}{2} }\right)}^{2m + \alpha}]$$

$$f(x,y,z) = 3y^2z \left( 3+\frac{7x+5}{1+y^2} \right)$$

Fraction

\frac{1}{3} or 1 \over 3

$$\frac{1}{3}$$

\sqrt{2} or \sqrt[n]{3}

$$\sqrt[n]{3}$$

$$f(x1,x2,\ldots,xn) = x1^2 + x2^2 + \cdots + xn^2$$

$$\vec{a} \cdot \vec{b}=0$$

Integral

$$\int_0^1 x^2 {\rm d}x$$

CIE 1931(x,y) chromaticity coordinates

Posted on 2016-01-23   |  

The chromaticity coordinates (x, y) can be calculated with spectrum and CIE 1931 xyz color matching functions.

$X=\int_0^\infty I(\lambda)\overline{x}(\lambda){\rm d}\lambda$

$Y=\int_0^\infty I(\lambda)\overline{y}(\lambda){\rm d}\lambda$

$Z=\int_0^\infty I(\lambda)\overline{z}(\lambda){\rm d}\lambda$

where $I(\lambda)$ is the spectrum distribution. And (x, y)is:

x = X/(X+Y+Z)
y = Y/(X+Y+Z)

There is another parameter “z”, which represents the lightness and can be given as:

z = 1-x-y

Photometry

Posted on 2016-01-20   |  

Photometry is the science of the measurement of light, in terms of its perceived brightness to the human eye.
– Wikipedia: https://en.wikipedia.org/wiki/Photometry_(optics))

The most accurate and comprehensive parameter, in my opinion, to describe the light property is spectrum. While in the practical applications, the spectrum information is not intuitive, thus the photometric parameters come out. As defined by Wikipedia, the photometry is in terms of the perceived brightness to the human eye, thus it represents the feeling of human to the light, which is more simple and intuitive.
　　The first parameters are “Tristimulus Value” defined by CIE in 1931. The value is to define the response of human eyes to three kinds of light on different wavelength. The color matching functions are denoted as r-bar, g-bar and b-bar, as shown in the figure below. Detailed data please refer to my github repo: https://github.com/zengxu/pyphotometrics

Photometirc tool by Python

Posted on 2016-01-20   |  

The project originates from one request from my colleague. One day, I received an Email to ask the password of one software which has been used to calculate the photometric characteristics of light. The software was developed by previous colleagues several years ago and nobody remembered the password. Then one idea “appears” in my mind, why don’t I develop a software tool and share it to others?
　　I roughly searched relevant information via Google. As it was an old subject and there were many tools to calculate the photometrics of light, both commercial available and open source. However, according to my knowledge, there was no open source written by Python. Personally, I think Python is a very good and easy-to-use programming language, thus I plan to develop such kind of Photometric tool written by Python and share the source code to others via Github.

Goal: Develop a tool to calculate at least the basic photometric parameters of light.
Programming language: The tool will be written by Python, all the source code will be shared freely via Github.
Duration: Two months, to be finished before March 31st.

Debug to Failed connect to github.com:443; No error

Posted on 2016-01-19   |  

Reason: Internet connection failed
Solution: Set proxy in .gitconfig file
Steps:

  1. Open .gitconfig file under C:\Users\310038541…

  2. Edit .gitconfig file. Add the following script at the end of the file:

    [http]
    　　proxy = http://www.proxy.com:8080/

    Please note: a “\tab” should be added before the second line, replace the proxy website by your own proxy IP address
